# MCP Gateway Desktop

A desktop application for managing your MCP Gateway configuration, servers, and groups.

## Features

- **Marketplace**: Browse and add servers from the official MCP registry.
- **Group Management**: Organize servers into groups.
- **Configuration**: Visual editor for your `config.json`.

## Development

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run in development mode:
   ```bash
   npm run dev
   ```

## Build

To build the application for production:

```bash
npm run build
```

The output will be in the `dist` and `dist-electron` directories.
