# MCP Gateway

Aggregate multiple Model Context Protocol (MCP) servers into a single endpoint.

## Projects

- **[Desktop App](./desktop)**: A visual interface for managing your gateway.
- **[CLI / Server](./package)**: The core gateway server and CLI tool.
- **[Documentation](./docs)**: The documentation website.

## Getting Started

Check the [Documentation](./docs) for full guides.

### Quick Start (CLI)

```bash
npm install -g @mcp-gateway/cli
mcp-gateway init
```
